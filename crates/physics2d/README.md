# vyxen-physics2d

**Physics types for the Vyxen library in 2d development**

</div>

vyxen-physics2d exposes multiple types such as
 - RigidBody
 - SoftBody
 - Spring
 - PointMass
 - Collision
 - ContactPoints
 - Manifold

This library is included automatically when you import Vyxen.
